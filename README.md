# Vivian Bakes

Portfolio and contact site for Vivian Bakes, a fine art cake maker based in London, Ontario.

**Live site:** [vivianbakes.com](https://vivianbakes.com)

## Pages

- **Home** — Hero, category cards, featured work bento grid, about teaser, and contact CTA
- **Portfolio** — Full masonry gallery of cakes, cupcakes, and confections with lightbox viewer
- **About** — Vivian's background and story
- **Contact** — Inquiry form (via Formspree) with event type and date fields

## Tech

- Static HTML/CSS/JS — no build step, no framework
- Hosted on GitHub Pages with custom domain
- [Fraunces](https://fonts.google.com/specimen/Fraunces) (headings) + [Outfit](https://fonts.google.com/specimen/Outfit) (body) via Google Fonts
- [GLightbox](https://biati-digital.github.io/glightbox/) for image lightbox
- Images in WebP, resized to max 1200px wide

## Structure

```
index.html          Home page
portfolio.html      Full gallery
about.html          About page
contact.html        Contact form
css/style.css       All styles (mobile-first, responsive)
js/main.js          Shared JS (fade-in, lightbox, mobile nav)
images/             Optimized WebP images
favicon.svg         Site favicon
CNAME               Custom domain config
```
